== Description ==

VisitorsCafe Video Chat is a plugin that Empowers members of your community by allowing visitors on your site to connect with each other and socialize live around your content.

== Installation ==

1. Upload `visitorscafe-video-chat.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to General Setting page and insert the code
4. Save the changes
